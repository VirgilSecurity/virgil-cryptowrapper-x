Asn1Writer

@objc(VSCFAsn1Writer)
public protocol Asn1Writer : CContext

Provides interface to the ASN.1 writer. Note, elements are written starting from the buffer ending. Note, that all “write” methods move writing position backward.

  • reset(out:outLen:)

    Reset all internal states and prepare to new ASN.1 writing operations.

  • finish(doNotAdjust:)

    Finalize writing and forbid further operations.

    Note, that ASN.1 structure is always written to the buffer end, and if argument “do not adjust” is false, then data is moved to the beginning, otherwise - data is left at the buffer end.

    Returns length of the written bytes.
